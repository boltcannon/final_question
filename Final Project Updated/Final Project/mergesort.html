Mergesort – Divide, Sort & Merge

How to use:
1. Click Divide on the main array to split it into two halves. The original array box stays as a black empty frame.
2. Two learners can now work in parallel on the left and right arrays: divide them further, then sort inside each part.
3. When you are ready to merge, drag numbers from the child arrays into the empty slots of the parent frame in sorted order.
4. Continue until you have a fully sorted array at the top.
5. Use Check if Sorted to see if the main array (top frame) is in ascending order.

Supports multi-touch: multiple users can drag different numbers at the same time on a touch table.
