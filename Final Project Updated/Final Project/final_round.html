📋 The Problem

A school has 6 students. Their test scores (out of 50) are mixed up and need to be sorted and searched efficiently.

Your team must complete 4 tasks in order:
🔵 Task A — Sort the scores using Merge Sort
🟢 Task B — Sort the scores using Bubble Sort (compare the two approaches)
🟣 Task C — Search for score 25 using Linear Search
🟠 Task D — Search for score 25 using Binary Search (on the sorted list)
💡 General Hint:
• Merge Sort works by splitting the list in half repeatedly, then merging halves back in sorted order.
• Bubble Sort works by comparing two neighbors at a time and swapping them if out of order.
• Linear Search checks every element one by one — no sorting needed.
• Binary Search jumps to the middle and cuts the search in half — but ONLY works on a sorted list!
Work together and use the step cards to reason through the correct order.
